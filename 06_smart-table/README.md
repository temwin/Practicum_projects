# 🧾 Smart Table (Работа с API)
Интерактивная таблица с фильтрацией, поиском, сортировкой и пагинацией, теперь работающая с сервером через API.
Цель проекта: научиться адаптировать существующий код под работу с асинхронными источниками данных, использовать параметры запроса и реализовать полноценное обновление интерфейса ⚡.

## 🛠 Что было сделано
- Переведена таблица с локальных данных на работу с серверным API
- Реализованы асинхронные функции для получения индексов и записей (getIndexes, getRecords)
- Настроена пагинация с сервера и обновление интерфейса после получения данных
- Фильтрация и поиск теперь работают через сервер, параметры формируются в объект запроса
- Сортировка применена на серверной стороне, query-параметры передаются через field:order
- Упрощена архитектура компонентов: каждая функциональность (фильтр, поиск, сортировка, пагинация) возвращает функции для применения параметров и обновления UI
- Поддержка кеширования запросов и повторного использования параметров запроса

## 🚀 Запуск проекта
1. Склонируйте репозиторий:  
   ```bash
   git clone https://github.com/temwin/Practicum_projects.git
   ```
2. Откройте файл index.html в браузере
3. Проверьте работу таблицы: поиск, фильтрация, сортировка и пагинация должны корректно работать с сервером
