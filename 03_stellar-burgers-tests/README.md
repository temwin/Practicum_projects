# 🍔 Stellar Burgers (Космическая бургерная)
Интерактивное приложение для сборки космических бургеров с возможностью перетаскивания ингредиентов, оформления заказов и личным кабинетом. Проект покрыт юнит-тестами (Jest) и интеграционными тестами (Cypress) для обеспечения надёжности ключевых сценариев.
Цель проекта: освоить написание тестов для React-приложений, научиться мокировать данные и запросы к API, покрыть тестами редьюсеры и пользовательские сценарии 🧪

## 🛠 Что было сделано
- Настроены тестовые окружения: Jest для юнит-тестирования и Cypress для интеграционных тестов
- Созданы моковые данные для ингредиентов и ответов сервера (заказы, пользователь)
- Написаны интеграционные тесты (Cypress) для страницы конструктора:
- Добавление ингредиентов в конструктор (булки и начинки)
- Открытие и закрытие модального окна ингредиента (по крестику и оверлею)
- Создание заказа: авторизация, сборка бургера, проверка номера заказа, очистка конструктора
- Написаны юнит-тесты (Jest) для редьюсеров:
- Проверка инициализации rootReducer
- Тестирование слайса burgerConstructor: добавление, удаление, сортировка ингредиентов
- Тестирование слайса ingredients: обработка экшенов Request/Success/Failed (изменение состояния загрузки, сохранение данных, запись ошибки)
- Все тесты успешно проходят и покрывают ключевую функциональность приложения

## 🚀 Запуск проекта
1. Склонируйте репозиторий:
```bash
git clone https://github.com/temwin/Practicum_projects.git
```
2.Перейдите в директорию проекта:
```bash
cd Practicum_projects/03_stellar-burgers-tests
``` 
3. Установите зависимости:
```bash
npm install
# или
yarn
```
4. Запустите проект в режиме разработки:
```bash
npm run start
# или
yarn start
```

## 🧪 Запуск тестов
1. Юнит-тесты (Jest):
```bash
npm run test
# или
yarn test
```
2. Интеграционные тесты (Cypress):
```bash
npm run cypress
# или
yarn cypress
```
3. Затем выберите Component testing → браузер (например, Chrome) → файл contentForm.cy.tsx
