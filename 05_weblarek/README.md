# 🛒 Веб-ларек (Интернет-магазин для разработчиков)
Интерактивный интернет-магазин с каталогом товаров, корзиной и оформлением заказов. Проект построен на архитектуре MVP (Model-View-Presenter) с событийно-ориентированным взаимодействием между слоями.
Цель проекта: освоить организацию кода по шаблону MVP, реализовать полный цикл покупки — от отображения товаров до отправки заказа на сервер, и научиться управлять состоянием приложения через события ⚡.

## 🛠 Что было сделано
- Реализована архитектура MVP: модели данных (ProductCatalog, Cart, Customer), слои представления (компоненты интерфейса) и презентер, связывающий их через брокер событий EventEmitter
- Создан базовый класс Component для всех UI-компонентов, упрощающий рендеринг и обновление DOM
- Разработаны модели для управления товарами, корзиной и данными покупателя с генерацией событий при изменениях
- Реализован слой коммуникации Communication для взаимодействия с сервером (получение товаров, отправка заказа)
- Созданы компоненты представления: галерея товаров, карточки (каталог, превью, корзина), модальные окна, формы (заказа и контактов), корзина, сообщение об успешном заказе
- Настроена обработка событий в презентере (main.ts), который реагирует на изменения моделей и действия пользователя, обновляя интерфейс
- Реализована логика добавления/удаления товаров из корзины, подсчёт суммы, валидация форм, переходы между этапами оформления заказа

## 🚀 Запуск проекта
1. Склонируйте репозиторий:
```bash
git clone https://github.com/temwin/Practicum_projects.git
```
2.Перейдите в директорию проекта:
3. Установите зависимости:
```bash
npm install
# или
yarn
```
4. Запустите проект в режиме разработки:
```bash
npm run start
# или
yarn start
```
5. Откройте браузер по адресу, указанному в терминале (обычно http://localhost:5173)
6. Для сборки production-версии выполните:
```bash
npm run build
# или
yarn build
```
