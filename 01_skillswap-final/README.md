# 🤝 SkillSwap (Обмен навыками)
Одностраничное веб-приложение для взаимного обмена навыками между пользователями. Позволяет публиковать навыки двух типов: «Учу» (чем готов делиться) и «Учусь» (чему хочет научиться), находить взаимные совпадения и управлять заявками на обмен.

Цель проекта: освоить методологию Feature-Sliced Design (FSD) и создать масштабируемое приложение с чётким разделением ответственности ⚡

## 🛠 Что было сделано
- Реализована архитектура по методологии Feature-Sliced Design (FSD): разделение на слои `shared`, `entities`, `features`, `widgets`, `pages`, `app`
- Созданы функции для публикации навыков «Учу» и «Учусь», поиска взаимных совпадений
- Реализована система заявок на обмен навыками и ведение списка текущих/завершённых сессий
- Настроена фильтрация навыков других пользователей
- Добавлено сохранение избранного и настроек в браузере (localStorage)
- Проект написан на React + TypeScript с использованием Redux Toolkit для управления состоянием
- Настроены тесты (Jest) и документация компонентов (Storybook)

## 🚀 Запуск проекта
1. Склонируйте репозиторий:
```bash
git clone https://github.com/temwin/Practicum_projects.git
```
2. Перейдите в директорию проекта:
```bash
cd 01_skillswap-final
```
3. Установите зависимости:
```bash
npm install
```
4. Запустите проект в режиме разработки:
```bash
npm run dev
```

## 📁 Структура проекта (FSD)
**shared/ — общие UI-компоненты, хуки, утилиты**
**entities/ — сущности (навыки пользователей)**
**features/ — бизнес-логика (авторизация, выбор навыков)**
**widgets/ — крупные UI-компоненты (карточки, фильтры)**
**pages/ — страницы приложения**
**app/ — точка входа, провайдеры, роутинг**
