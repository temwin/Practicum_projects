# SkillSwap

## Веб-приложение для взаимного обмена навыками

## Описание проекта

SkillSwap — это одностраничное веб-приложение для обмена навыками между пользователями. В нем пользователи могут публиковать навыки двух типов:

- “Учу” — навыки, которыми они готовы делиться
- “Учусь” — навыки, которым хотят научиться.

Приложение позволяет находить взаимные совпадения, отправлять заявки на обмен навыками и вести список текущих/завершённых сессий.

## Основные функции

- Просмотр и фильтрация навыков других пользователей
- Создание собственных списков "Учу" и "Учусь"
- Поиск взаимных совпадений навыков
- Отправка и управление заявками на обмен
- Ведение списка текущих и завершенных сессий
- Сохранение избранного и настроек в браузере

## Структура проекта

Проект организован в соответствии с методологией Feature-Sliced Design (FSD) для обеспечения масштабируемости и четкости разделения ответственности.

- `shared/` - общие UI-компоненты, хуки и вспомогательные функции. Можно использовать в любом месте проекта.
- `entities/` - сущности приложения (например, навыки пользователей). Используется в Features, Widgets, Pages, App.
- `features/` - бизнес логика (авторизация, выбор навыков и т.п). Используется в Widgets, Pages, App.
- `widgets/` - крупные UI-компоненты (карточки, фильтры), из которых собираются страницы. Используется в Pages, App.
- `pages/` - страницы приложения, подключаемые в маршрутизации. Используется только в App.
- `app/` - точка входа, провайдеры, роутинг.
