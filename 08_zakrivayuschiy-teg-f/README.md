# 💻 Закрывающий тег

Интерактивный одностраничный сайт, где каждая карточка оживает при взаимодействии.
Цель проекта: научиться создавать анимированные элементы, работать с SVG и CSS-переменными, а также развивать адаптивные и интерактивные интерфейсы ⚡.

## 🎨 Что можно делать на сайте

Лайкать и сохранять карточки с анимацией сердечка ❤️

Открывать модальное окно с дополнительной информацией

Любоваться плавными градиентами, сеткой и декоративными элементами

Смотреть на адаптивный дизайн на любом экране — от телефона до десктопа

## 🛠 Технологии

HTML5 — семантика и структурирование

CSS3 — гриды, переменные, адаптивная вёрстка, псевдоэлементы

JavaScript — интерактивные кнопки и SVG-анимация

SVG — анимация сердечек и декоративных деталей

ARIA — доступность кнопок и диалогов
