# 🧘‍♂️ Сложно сосредоточиться

Адаптивный одностраничный сайт с поддержкой светлой и тёмной темы.  
Цель проекта: научиться верстать адаптивный сайт, использовать CSS-переменные, гриды, логические свойства и реализовать переключение тем с помощью JavaScript ⚡.

---

## 🎮 Как пользоваться

1. Просмотрите шапку сайта с заголовком и описанием темы.  
2. Используйте кнопки «День», «Авто» и «Неон» для переключения темы.  
3. Ознакомьтесь с секциями:
   - «Почему сосредоточиться так сложно»  
   - «Что снижает концентрацию внимания?»  
   - «Как концентрироваться лучше»  
   - «А можно в картинках?»  
4. Галерея с изображениями поддерживает ленивую загрузку (lazy loading).  
5. В футере отображается заголовок «focus» и декоративные элементы.

---

## 🛠 Стек технологий

- HTML5 — семантическая разметка  
- CSS3 — адаптивная верстка, CSS-переменные, гриды  
- JavaScript — переключение тем  
- ARIA — доступность декоративных элементов  
- Ленивая загрузка изображений через `loading="lazy"`  
- Адаптивный шрифт заголовков с `clamp()`  
- Псевдоэлементы для декоративных элементов
