# Сложно сосредоточиться

Проект «Сложно сосредоточиться» — это адаптивный одностраничный сайт с поддержкой светлой и тёмной темы.  
Цель проекта: научиться верстать адаптивный сайт, использовать CSS-переменные, гриды, логические свойства и реализовать переключение тем с помощью JavaScript.

## Функционал
- Адаптивная верстка по принципу **Mobile First**
- Поддержка **тёмной и светлой темы**
- Автоматическое переключение темы в зависимости от настроек системы пользователя
- Ручное переключение темы через кнопки «День», «Авто», «Неон»
- Статичная галерея с ленивой загрузкой изображений (`loading="lazy"`)
- Адаптивный шрифт заголовков с использованием `clamp()`
- Декоративные элементы через псевдоэлементы, скрытые от скринридеров

## Используемые технологии
- **HTML5** — семантическая разметка
- **CSS3** — адаптивная верстка, гриды, CSS-переменные
- **JavaScript** — переключение тем
- **ARIA** — доступность декоративных элементов
- **Ленивая загрузка** изображений через `loading="lazy"`

## Запуск проекта

