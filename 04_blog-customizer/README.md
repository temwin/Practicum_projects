# 📝 Блог с настраиваемой панелью (CSS Customization)
Интерактивная страница блога с выдвижной панелью настроек, позволяющей изменять внешний вид статьи через CSS-переменные. Проект построен на компонентном подходе с использованием React и Storybook для документирования UI-кита.
Цель проекта: освоить управление состоянием компонентов, организацию передачи данных между формой и страницей, реализовать кастомизацию через CSS-переменные и отработать навыки композиции компонентов ⚡.

## 🛠 Что было сделано
- Реализован выдвижной сайдбар с настройками оформления статьи (открытие/закрытие по клику на стрелку или вне области)
- Создана форма редактирования свойств ArticleParamsForm с использованием готовых UI-компонентов из storybook
- Организована композиция компонентов и продумана архитектура хранения состояния
- Реализована логика раздельного хранения состояния формы (временные настройки) и состояния страницы (применённые настройки)
- Настроено применение стилей к статье через CSS-переменные только после нажатия кнопки «Применить»
- Реализован сброс настроек к исходным значениям при нажатии кнопки «Сбросить»
- Обеспечено закрытие сайдбара при повторном нажатии на стрелку или клике вне области

## 🚀 Запуск проекта
1. Склонируйте репозиторий:
```bash
git clone https://github.com/temwin/Practicum_projects.git
```
2. Перейдите в директорию проекта:
```bash
cd Practicum_projects/04_blog-customizer
```
3. Установите зависимости:
```bash
npm install
# или
yarn
```
4. Запустите проект в режиме разработки:
```bash
npm run start
# или
yarn start
```
5. Для запуска Storybook (документация UI-компонентов):
```bash
npm run storybook
# или
yarn storybook
```
6. Для проверки кода используйте команды:
```bash
npm run lint        # линтер для JS/TS
npm run stylelint   # линтер для стилей
npm run format      # форматирование кода
```
