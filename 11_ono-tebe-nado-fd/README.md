#Оно тебе надо — аукцион необычных вещей

Интерактивная страница-аукцион, где представлены вещи, в которые никто не верил. Пользователь может просматривать лоты, узнавать подробности об аукционе и делать ставки.

##🎮 Как пользоваться

Просмотрите обложку с заголовком и кнопкой «Сделать ставку».

Изучите секцию с лотами: клик по карточке открывает детали лота.

Ознакомьтесь с секцией «Об аукционе» с описанием и логотипом.

В футере находятся контакты, меню и ссылки на соцсети.

##🛠 Стек технологий

HTML5, CSS3

Flexbox и CSS Grid

Pixel Perfect для точного соответствия макету

Подключение шрифтов и оверлеи для карточек

##🎨 Классы и структура

header — шапка, header__menu — меню, header__logo — логотип, header__address — контакты

cover — обложка, overlay — полупрозрачный слой, cover__title — заголовок, cover__description — подзаголовок и кнопка

lots — секция с лотами, lots__card-list — список карточек, card — отдельная карточка, overlay — оверлей внутри карточки

about — секция «Об аукционе», about__column — колонки, about__logo — логотип, about__text — текст

footer — подвал, footer__menu — ссылки, footer__social-list — соцсети

##🚀 Запуск проекта

Скачать Node.js — https://nodejs.org/

Открыть проект в VS Code

Подключить стили в index.html

Открыть index.html в браузере
