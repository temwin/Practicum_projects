# Посмотри в окно

Интерактивное веб-приложение для просмотра видео по городам и времени суток. Учебный проект Яндекс.Практикума по верстке и работе с готовыми модулями.

## 🎮 Как пользоваться

1. Введите город в поле ввода.
2. Выберите время суток с помощью чекбоксов.
3. Нажмите кнопку «Найти» для отображения видео.
4. Кликните на карточку видео, чтобы оно воспроизвелось в основном блоке.
5. Нажмите кнопку «Показать ещё», чтобы подгрузить дополнительные видео.
6. В случае ошибки появится сообщение о недоступности видео для выбранного города.

## 🛠 Стек технологий

- HTML5, CSS3
- Flexbox и CSS Grid
- Pixel Perfect
- Работа с шаблонами `<template>` для динамических элементов
- Стилизация форм, кнопок и состояний карточек
- Скрытие элементов через `.visually-hidden`

## 🚀 Запуск проекта

1. Скачать Node.js — [https://nodejs.org/](https://nodejs.org/)
2. Открыть проект в VS Code
3. Открыть `index.html` в браузере
