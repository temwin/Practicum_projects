# 🍔 Stellar Burgers (Космическая бургерная)
Космическая бургерная с возможностью сборки бургеров, оформления заказов и личным кабинетом. Проект построен на React, Redux и роутинге, покрыт юнит-тестами (Jest) и интеграционными тестами (Cypress).
Цель проекта: освоить полный цикл разработки React-приложения — от настройки роутинга и глобального состояния до авторизации и тестирования ключевых сценариев 🚀

## 🛠 Что было сделано
- Настроен роутинг для навигации по страницам приложения
- Реализованы запросы к серверу через Redux и глобальный store (ручки API вынесены в utils/burger-api.ts)
- Настроена авторизация и созданы защищённые роуты для личного кабинета

## 🚀 Запуск проекта
1. Склонируйте репозиторий:
```bash
git clone https://github.com/temwin/Practicum_projects.git
```
2. Перейдите в директорию проекта:
```bash
cd Practicum_projects/02_stellar-burgers  
```
3. Установите зависимости:
```bash
npm install
# или
yarn
```
4. Запустите проект в режиме разработки:
```bash
npm run start
# или
yarn start
```
