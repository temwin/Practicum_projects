🍪 Пряники
Интерактивная программа для анализа продаж в магазине: составляется рейтинг продавцов по доходности и рассчитываются бонусы для каждого. Цель проекта: научиться работать с данными на JavaScript, выделять чистые функции для обработки информации и строить модульную архитектуру ⚡.

🛠 Что было сделано
- Разработан скрипт на чистом JavaScript для анализа данных о продажах
- Выделены функции для расчета прибыли и бонусов, которые можно изменять независимо от основной логики
- Реализована обработка нескольких датасетов с динамической загрузкой
- Использованы чистые функции без глобальных переменных и побочных эффектов
- Подготовлена структура проекта для простого добавления новых датасетов и расширения функционала

📂 Структура проекта
- data/ — примеры датасетов (dataset_1.js, dataset_2.js, dataset_3.js)
- src/main.js — основной файл программы
-index.html — запускает скрипт в браузере

🚀 Запуск проекта
1. Склонируйте репозиторий:
git clone https://github.com/temwin/sales-bonus
2. Откройте файл index.html в браузере
В консоли DevTools (CTRL+Shift+I) будет выведен рейтинг продавцов и рассчитанные бонусы
